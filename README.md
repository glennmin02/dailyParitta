# Daily Paritta 🙏

Personal web app for Myanmar Buddhist daily prayers with offline support.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5174

## Features

- 11 Daily Burmese Buddhist prayers
- Individual counter for each prayer (unlimited)
- Offline support (PWA)
- Multi-language display (Burmese, romanization, English)
- Dark/light themes
- Auto-save progress

## Prayers Included

1. နတ်ပင့်ဂါထာ - Inviting the Devas
2. ဩကာသ ကန်တော့ချိုး - Requesting Permission
3. သရဏဂုံ သုံးပါး - Triple Gem Refuge
4. ဘုရားဂုဏ်တော် ၉ ပါး - 9 Buddha Virtues
5. တရားဂုဏ်တော် ၆ ပါး - 6 Dhamma Virtues
6. သံဃာဂုဏ်တော် ၉ ပါး - 9 Sangha Virtues
7. ဂုဏ်တော်ကွန်ချာ ၁၈ ထပ် - 18 Vajira Pakara Levels
8. သမ္ဗုဒ္ဓေ ဂါထာ - 28 Buddha Salutation
9. မဟာသမယသုတ် - Maha Samaya Sutta
10. စဓ ဗဝ ဂါထာ - Blessings Gatha
11. နတ်အပို့ဂါထာ - Sending Devas Back

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4
- Zustand (state)
- PWA support

## Build for Production

```bash
npm run build
npm run preview
```

## Install as PWA

### Mobile
1. Open in browser
2. Add to Home Screen
3. Use like native app

### Desktop
1. Chrome/Edge install icon
2. Click "Install"
